2. 训练过程

2.1 训练参数配置

  • 在DeepLearning的训练过程中，有许多需要作为超参数进行设置，使得下一次训练的时候非常方便直接便可以进行修改训练
  • 训练过程中需要配置的参数:
配置参数 缩写 描述
数据地址 —data_path 存放数据集的地址
设备 —gpu 设置使用gpu的卡数
训练轮数 —epoch 设置循环训练的次数
批大小 —batchsize 设置每个epoch的大小，一般4的倍数
输出文件地址 —out_path 训练记录,每个epoch训练模型
学习率 —lr 设置初始化学习率
优化方式 —optimizer 设置优化方式Adam等
权重衰减 —weight_decay 设置权重衰减参数，一般4e-5
加载模型 —checkpoint 设置重新训练所需要加载的模型

4.遇到的坑

4.1 数据处理常见坑

4.1.1 collate_fn必须要是tensor，list等，不能是object

  • 对于可变长度的输入，pytorch会默认把这个东西进行压缩，直接cat起来，而这样的话，就会报错
  • 处理办法: 将batchsize设为1,将collate_fn设为自己的函数

4.2 训练过程常见坑

4.2.1 Pytorch计算流图问题

  • pytorch无论是eval还是train都会创建计算流图，因此test的时候记得加上.item()或者eval的时候加上with torch.no_grad()
  • 误差图计算的时候很容易忽略掉初始化的item操作，要注意

4.2.2 Python程序效率问题

  • 少用循环，特别是在训练过程数据输入数据的时候，减少训练时间
