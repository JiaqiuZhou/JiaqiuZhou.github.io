1.Text editing image paper

1.1 Tell, Draw, and Repeat: Generating and Modifying Images Based on Continual Linguistic Instruction

值得借鉴: 通过迭代生成网络的方式来生成图片，每一阶段都会借鉴于前一阶段的语句特征向量

1.1.1 基本概念—Generative Neual Visual Artist(GeNeVA)

  • 解决问题: 当人们对人工智能生成的图片不满意的时候，可以通过语句输入来调整结果，

  • 方法简介: 结合GAN和图像Encoder以及语言处理GRU

  • 本文提出: 通过实验证明迭代完成任务比不迭代的方法好

  • 问题定义: 每一轮迭代包含一个Teller和一个Drawer，Teller通过语句和提供前一轮的画布来给Drawer，让Drawer优化绘画内容

2.2 2019Papers

值得借鉴: 从magezine布局生成的方法来看，貌似使用GAN来生成layout也是不错的方法

2.2.1 MirrorGAN: Learning Text-to-image Generation by Redescription

  • 基本概念: 提出一个网络从text生成图片，再从图片生成text来保证语义一致性

  • 网络结构:

  • 方法简述: 分成三个模块:STEM模块用于生成word和sentence-level的特征向量，GLAM用于从粗到细生成图片，STREAM用于重新生成语句描述来对齐给定的语句描述

    • STEM: Semantic Text Embedding Module,利用RNN来处理描述的text，利用conditioning augmentation来优化提出的特征向量

  • GLAM: Global-Local collaborative Attentive Module，通过多个阶段生成图像的模块

  • STREAM: Semantic Text REgeneration and Alignment Aodule,通过RNN模块重新生成text的描述

  • Loss函数:训练GAN模块的时候考虑了自己还有STREAM的loss

2.2.2 Automatic Layout Generation for Graphical Design Magazines

    • 手动设计生成layout的规则
