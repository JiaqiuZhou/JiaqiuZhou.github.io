介绍在讲座中所学到的一些知识

1. 医疗

1.1 话题

  • 关于检测癌症，肿瘤分割，以判断是良性肿瘤还是恶性肿瘤

1.2 问题

  • 医疗中最主要的一个问题还是医生标注的一个问题，得不到标注就会使数据没法得到扩充，小样本数据问题。
  • 数据来源于不同的扫描仪，需要一定的预处理知识使得数据分布相近

2. 视觉识别

2.1 话题

  • 行人重识别:判断视频序列中是否存在特定行人，即对同一个行人进行ID标注
  • www.pkuvmc.com/slzhang.jdl@pku.edu.cn

2.2 问题

  • 训练数据的标注问题
  • 类内变化大如视角，光照，摄像机参数等差异比较大;类间差异小如不同人长相相似
  • 采集图像的分辨率太低，质量差

2.3 方案

  • 类内变化大:通过属性归类，比如说行人的衣服，脸型等属性的相关性来提高ID识别的准确性

  • 数据标注问题: 通过迁移学习的方法，固定人的pixel，去除背景，利用GAN重新生成背景再与人结合

  • 鲁棒特征:

    1）细节判别力: 网络可能只学到分类的特征，不够细节，从classification—>part classfication

    ​ 具体而言，利用CNN的feature maps来做聚类，进行part classification

    2) 姿态鲁棒性: part —> transformer/person—> pose nomalization —> part

  • 时空特征学习: 利用局部特征 1)动作和速度 2)动作的模板识别

3. 数据预测

  • Matrix Completion/Matrix Recovery/傅立叶变换

3.1 多目标追踪

  • Particle Filter based Tracking:粒子滤波/一步步提高权重

  • Classification For tracking: 优点:解决遮挡问题/缺点:分类错误的话，会导致后面错误跟踪漂移

  • Sparse Tracking

  • Correlation Filter

4. 模仿学习

4.1 话题

  • 从一个视频序列中学习得到一系列动作; input:视频 output:一个动作序列

  • 初衷: 1)更小的surpervision 2)更多的处理数据 3) 跟人类的行为有相似的地方

5. Semantic and Generative Deep Learning of 3D Object shapes

5.1 Stories of Generative surface reconstruction

  • Solution1: Picture—>meso(skeleton)—>mesh—>surface(Residual MeshNet: MLP)

  • Solution2: Picture—> voxel—> 3D 卷积(Topology modification Networks)

    • Normal loss(法向量)

    • CD loss:衡量mesh与GT之间的匹配程度; cd loss 原是用来衡量point cloud的，如果把cd loss 的点趋于无穷的话就可以用来衡量 mesh

    • error loss:真实距离是否在范围内/smooth loss:删面/boundary loss:边界平滑

5.2 Amodal 3D object Detection

  • Solution1: Frustum based methods Frustum ConvNet
  • Solution2: Hotspot Networks

6. 图像质量评估(Image Quality Assessment)

6.1 Introduction

  • IQA概念: 希望通过客观的指标来模拟人对图像质量的主观感受
  • Signal-level Evaluation: 峰值信噪比PSNR，通过衡量每个像素的峰值信噪比来评价，但是这种与人的感受有差别，因为人类通常对于图像比较注重于纹理和部分，而不是全部
  • SSIM: 通过关注图像的结构相似性，在边缘与纹理进行比较，与人的感受比较一致，也用的比较多，但不是最好的

6.2 方法分类

  • 参考图像: 根据有无参考图像可以分为几类，一类是Fully-reference，也就是完全有参考图像的，一类是Side-info reference，也就是半参考图像，还有一种就是No-reference
  • 美学标准: 拍摄规则，焦距，对称性等等

6.3 Application

  • 美学角度: 广告海报，封面筛选，相册管理，照片美学管理，构图推荐，图像自动裁剪，人脸美学增强，美学评估
  • 质量角度: 视频清晰度，视频连续性，压缩性能

6.4 Discussion

  • 小样本泛化性能问题，IQA新应用，主观性评价问题
